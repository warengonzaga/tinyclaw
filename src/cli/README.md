# tinyclaw

Command-line interface for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Routes `tinyclaw` commands (setup, start, config, seed, backup, purge) with a lightweight argv-based router.

## Installation

```bash
bun add -g tinyclaw
```

## Usage

```bash
tinyclaw setup    # First-time setup
tinyclaw start    # Start the agent
tinyclaw config   # Manage configuration
tinyclaw backup   # Backup agent data
tinyclaw purge    # Reset agent data
```
