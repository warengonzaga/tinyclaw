# @tinyclaw/matcher

Hybrid semantic text matching engine for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides a three-dimensional text matcher combining keyword overlap (TF-IDF), fuzzy Levenshtein distance, and synonym expansion — all offline, no external APIs needed.

## Installation

```bash
bun add @tinyclaw/matcher
```
