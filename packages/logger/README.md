# @tinyclaw/logger

Context-aware structured logging for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides a pre-configured `LogEngine` wrapper with custom emoji mappings, runtime log-mode switching, and the shared `logger` singleton.

## Installation

```bash
bun add @tinyclaw/logger
```

This package is part of the [Tiny Claw](https://github.com/warengonzaga/tinyclaw) monorepo.
