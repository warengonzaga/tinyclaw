# @tinyclaw/secrets

Machine-bound encrypted secrets management for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `SecretsManager`, secrets tools, and key-name builders for storing API keys and credentials with AES-256-GCM machine-bound encryption.

## Installation

```bash
bun add @tinyclaw/secrets
```
