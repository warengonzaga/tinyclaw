# @tinyclaw/pulse

Cron-like recurring task scheduler for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `createPulseScheduler` — a lightweight interval-based scheduler supporting human-readable schedules ('30m', '1h', '24h') that runs handlers through the session queue.

## Installation

```bash
bun add @tinyclaw/pulse
```
