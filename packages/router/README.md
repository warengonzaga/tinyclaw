# @tinyclaw/router

Multi-provider LLM routing and query classification for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `ProviderOrchestrator`, query classifier (tier-based), and provider registry for intelligent routing across multiple LLM providers.

## Installation

```bash
bun add @tinyclaw/router
```
