# @tinyclaw/learning

Pattern detection and adaptive learning engine for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `createLearningEngine` which detects signals from conversations, stores weighted patterns to disk, and provides learned context for agent adaptation.

## Installation

```bash
bun add @tinyclaw/learning
```
