# @tinyclaw/core

Core agent runtime for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides the main agent loop, SQLite database factory, built-in Ollama LLM provider, model constants, onboarding messages, and TOTP/crypto owner-auth utilities.

## Installation

```bash
bun add @tinyclaw/core
```
