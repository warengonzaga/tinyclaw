# @tinyclaw/heartware

Personality and self-configuration system for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `HeartwareManager`, heartware tools, context/shield/memory loaders, soul generator for trait generation from seeds, and creator meta fetcher.

## Installation

```bash
bun add @tinyclaw/heartware
```
