# @tinyclaw/queue

Per-session promise-chain task serialization for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `createSessionQueue` which serializes tasks per user using a promise-chain pattern, allowing parallel execution across different users with zero dependencies.

## Installation

```bash
bun add @tinyclaw/queue
```
