# @tinyclaw/config

SQLite-backed persistent configuration for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `ConfigManager`, config tools, and Zod-validated schemas for agent settings, provider preferences, and channel configs.

## Installation

```bash
bun add @tinyclaw/config
```
