# @tinyclaw/shell

Controlled shell execution with permission engine for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides shell engine, permission engine (allowlist/blocklist/approval store), shell executor with safety controls, and agent tools — with Shield integration and owner-only enforcement.

## Installation

```bash
bun add @tinyclaw/shell
```
