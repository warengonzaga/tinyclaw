# @tinyclaw/delegation

Sub-agent delegation and lifecycle management for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides sub-agent runners, orientation context builder, lifecycle manager, template manager, background runner, and delegation tool factory.

## Installation

```bash
bun add @tinyclaw/delegation
```
