# @tinyclaw/shield

Runtime SHIELD.md threat evaluation and enforcement for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides `createShieldEngine`, SHIELD.md parser, and event matcher for deterministic threat-based blocking/allowing of agent actions at runtime.

## Installation

```bash
bun add @tinyclaw/shield
```
