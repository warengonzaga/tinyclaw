# @tinyclaw/intercom

Lightweight pub/sub event bus for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides a topic-based event bus with wildcard subscriptions and bounded ring-buffer history covering task, agent, memory, and blackboard events.

## Installation

```bash
bun add @tinyclaw/intercom
```
