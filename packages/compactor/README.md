# @tinyclaw/compactor

Layered conversation compaction and token optimization for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides a multi-layer compression pipeline including rule-based pre-compression, shingle deduplication, LLM summarization, tiered summaries, dictionary encoding, and tokenizer optimization.

## Installation

```bash
bun add @tinyclaw/compactor
```
