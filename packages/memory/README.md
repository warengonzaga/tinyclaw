# @tinyclaw/memory

3-layer adaptive memory with temporal decay for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides an episodic memory engine with full-text search (BM25), Ebbinghaus temporal decay, and importance scoring — 100% local, no external embedding APIs.

## Installation

```bash
bun add @tinyclaw/memory
```
