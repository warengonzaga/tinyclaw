# @tinyclaw/sandbox

Lightweight sandboxed code execution for [Tiny Claw](https://github.com/warengonzaga/tinyclaw) — your autonomous AI companion.

Provides a code sandbox using Bun Worker threads with permission guards (no net/fs by default), configurable timeouts, and blocked dangerous globals.

## Installation

```bash
bun add @tinyclaw/sandbox
```
